## Containers
- Vector
- Map
- Stack
- Set

For Map and Set, the Read-Black Tree was re-implemented as well.

## Other implementations
- std::iterator_traits
- std::reverse_iterator
- std::enable_if
- std::is_integral
- std::equal and/or std::lexicographical_compare
- std::pair
- std::make_pair
